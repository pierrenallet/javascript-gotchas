What are JavaScript Gotchas?

The JavaScript is a simple, friendly, and forgiving programming language. This is one of the reasons it has become so very popular as the web has grown.

The forgiving and simple nature of JavaScript is usually a benefit. But there are many surprises and even bugs hiding in plain sight in most code. JavaScript gotchas are the most common of these bugs and bumps that you hit as you learn and write JavaScript.

We built this site and online training course to help new and experienced developers avoid these common but subtle bugs.
